  1. POST
    post() 方法可以发送 POST 请求到指定 url，一般格式如下：
  • url 请求 url。
  • data 参数为要发送到指定 url 的字典、元组列表、字节或文件对象。
  • json 参数为要发送到指定 url 的 JSON 对象。
  • args 为其他参数，比如 cookies、headers、verify等。

字典类型:

import requests

# 定义目标 URL
url = 'https://httpbin.org/post'

# 定义要发送的数据
# data 是一个字典，它将作为表单数据发送
data = {'key1': 'value1', 'key2': 'value2'}

# 发送 POST 请求
# response 是响应对象，你可以通过 response.text 来查看响应的文本内容
response = requests.post(url, data=data)

# 输出响应内容
print(response.text)

Json类型：

import requests
import json

# 定义目标 URL
url = 'https://httpbin.org/post'

# 定义要发送的 JSON 数据
json_data = {'key1': 'value1', 'key2': 'value2'}

# 发送 POST 请求
response = requests.post(url, json=json_data)

# 输出响应内容
print(response.text)

超时设置：
有时请求可能因为网络问题而长时间没有响应。你可以通过设置 timeout 参数来限制请求的最长等待时间：

import requests

try:
    response = requests.post('https://httpbin.org/post', data={'key': 'value'}, timeout=5)
    print(response.text)
except requests.exceptions.Timeout:
    print('请求超时')

在这个例子中，timeout 参数设置了请求的最大等待时间（以秒为单位）。如果超出这个时间，requests 会抛出 requests.exceptions.Timeout 异常。

认证：
有时你需要进行 HTTP 认证。requests 库提供了简单的基本认证和更复杂的认证方法：
基本认证：

from requests.auth import HTTPBasicAuth

response = requests.post('https://httpbin.org/post', data={'key': 'value'}, auth=HTTPBasicAuth('username', 'password'))
print(response.text)

OAuth 认证：
对于更复杂的认证（如 OAuth），你可以手动设置头部：

import requests
headers = {'Authorization': 'Bearer your-token'}
response = requests.post('https://httpbin.org/post', data={'key': 'value'}, headers=headers)
print(response.text)

代理设置：
如果你需要通过代理发送请求，可以使用 proxies 参数：

import requests
proxies = {
    'http': 'http://user:password@proxyserver:port',
    'https': 'http://user:password@proxyserver:port'
}

response = requests.post('https://httpbin.org/post', data={'key': 'value'}, proxies=proxies)
print(response.text)

数据编码;
有时你需要对发送的数据进行特殊的编码。requests 库默认使用 application/x-www-form-urlencoded 编码。如果你需要其他编码类型，可以使用 data 参数传递编码后的数据。
例如，发送 application/x-www-form-urlencoded 数据：

import requests
from urllib.parse import urlencode

data = {'key1': 'value1', 'key2': 'value2'}
encoded_data = urlencode(data)
response = requests.post('https://httpbin.org/post', data=encoded_data, headers={'Content-Type': 'application/x-www-form-urlencoded'})
print(response.text)

使用 Session 对象
requests.Session 对象可以用来在多个请求之间保持某些设置，比如 cookies、头部、认证信息等：

import requests

session = requests.Session()

# 设置通用的头部
session.headers.update({'User-Agent': 'my-app'})

# 发送一个请求
response = session.post('https://httpbin.org/post', data={'key': 'value'})
print(response.text)

# 发送另一个请求，保持相同的会话设置
response = session.get('https://httpbin.org/get')
print(response.text)

发送自定义请求体
你可以使用 data 或 json 参数传递请求体的内容，取决于你发送的数据格式：

使用 data 发送表单数据或 URL 编码的数据。
使用 json 发送 JSON 数据。
如果你需要更灵活地构造请求体，可以使用 data 参数传递原始字符串或自定义对象：

import requests

# 发送原始字符串作为请求体
response = requests.post('https://httpbin.org/post', data='raw data string')
print(response.text)

# 发送自定义对象
class CustomData:
    def __str__(self):
        return 'custom data'

response = requests.post('https://httpbin.org/post', data=CustomData())
print(response.text)
  1. GET

简单示例：

import requests

# 定义目标 URL
url = 'https://httpbin.org/get'

# 发送 GET 请求
#response 是响应对象，你可以通过 response.text 来查看响应的文本内容
response = requests.get(url)

# 输出响应内容
print(response.text)

传递查询参数
在 GET 请求中，通常需要传递查询参数。你可以使用 params 参数来添加这些查询参数

import requests

# 定义目标 URL
url = 'https://httpbin.org/get'

# 定义查询参数
params = {'key1': 'value1', 'key2': 'value2'}

# 发送 GET 请求
response = requests.get(url, params=params)

# 输出响应内容
print(response.text)

requests.get 方法返回一个 Response 对象，你可以通过它来获取响应的各种信息

import requests

# 发送 GET 请求
response = requests.get('https://httpbin.org/get')

# 响应状态码
print(response.status_code)

# 响应头部
print(response.headers)

# 响应内容（文本）
print(response.text)

# 响应内容（JSON）
print(response.json())

response.status_code 获取响应的状态码。
response.headers 获取响应的头部信息。
response.text 获取响应的内容（文本）。
response.json() 尝试将响应内容解析为 JSON 格式。

发送自定义头部
你可以使用 headers 参数添加自定义的 HTTP 头部：

import requests

# 定义目标 URL
url = 'https://httpbin.org/get'

# 定义自定义头部
headers = {'User-Agent': 'my-app'}

# 发送 GET 请求
response = requests.get(url, headers=headers)

# 输出响应内容
print(response.text)

处理请求失败
除了处理超时，你还可能需要处理其他类型的请求失败，如网络连接错误或服务器错误。你可以使用异常处理机制来处理这些错误：

import requests

try:
    response = requests.get('https://httpbin.org/get')
    response.raise_for_status()  # 如果响应状态码是 4xx 或 5xx，会抛出异常
    print(response.text)
except requests.exceptions.HTTPError as err:
    print(f'HTTP 错误发生：{err}')
except requests.exceptions.RequestException as err:
    print(f'请求错误发生：{err}')

response.raise_for_status() 方法会在响应状态码表示错误时抛出异常。requests.exceptions.RequestException 是所有 requests 库异常的基类，可以捕获所有请求相关的错误。

使用 Session 对象
requests.Session 对象可以用来在多个请求之间保持某些设置，如 cookies、头部等：

import requests

session = requests.Session()

# 设置通用的头部
session.headers.update({'User-Agent': 'my-app'})

# 发送一个 GET 请求
response = session.get('https://httpbin.org/get')
print(response.text)

示例：获取带有身份认证的内容
有时你需要进行身份认证来访问某些受保护的内容。你可以通过在 URL 中包含认证信息或使用 auth 参数来实现：

from requests.auth import HTTPBasicAuth

# 发送带有基本认证的 GET 请求
response = requests.get('https://httpbin.org/basic-auth/user/password', auth=HTTPBasicAuth('user', 'password'))
print(response.text)

在这个例子中：
auth 参数使用 HTTPBasicAuth 对象来提供基本认证信息。

示例：使用代理
如果你需要通过代理发送请求，可以使用 proxies 参数：

import requests

proxies = {
    'http': 'http://user:password@proxyserver:port',
    'https': 'http://user:password@proxyserver:port'
}

response = requests.get('https://httpbin.org/get', proxies=proxies)
print(response.text)

在这个例子中，proxies 是一个字典，其中 http 和 https 关键字的值分别是 HTTP 和 HTTPS 代理的地址。

<?php
function getRealIP() {
    if (!empty($_SERVER['HTTP_CLIENT_IP'])) {
        // 从 HTTP_CLIENT_IP 获取 IP
        $ip = $_SERVER['HTTP_CLIENT_IP'];
    } elseif (!empty($_SERVER['HTTP_X_FORWARDED_FOR'])) {
        // 从 HTTP_X_FORWARDED_FOR 获取 IP
        $ip = $_SERVER['HTTP_X_FORWARDED_FOR'];
    } else {
        // 从 REMOTE_ADDR 获取 IP
        $ip = $_SERVER['REMOTE_ADDR'];
    }

    // 处理 IP 可能是逗号分隔的列表
    $ipArray = explode(',', $ip);
    return trim($ipArray[0]);//trim除去空格，此处多此一举
}

echo ' IP Address: ' . getRealIP();
?>
